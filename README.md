# my-agent

🤖 我的第一个智能 Agent 项目

## 📖 项目简介

`my-agent` 是一个简单的智能代理（Agent）演示项目。旨在探索大语言模型（LLM）如何感知环境、进行推理并执行任务。作为我的第一个 Agent 项目，它实现了基础的对话交互和简单的工具调用功能。

## ✨ 主要功能

- **智能对话**：基于大语言模型的自然语言理解与生成。
- **工具调用**：能够根据用户指令调用外部工具（例如：搜索网络、计算、查询时间等）。
- **上下文记忆**：支持多轮对话，记住之前的交互内容。
- **易于扩展**：代码结构清晰，方便后续添加新的技能或工具。
